# e_commerce
Sert a une application web dédié a gerer un stock pour une magasin electrique. L application dois etre liée a une tablette fixe pour envoyer ou recevoir des quantité de fournisseur vers le stock , ainsi qu entre le stock vers le showRoom , et entre le showRoom et les clients.   Notre app dois etres fonctonnel, responsive , et sucurisé.
